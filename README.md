# Central Limit Theorem Simulation

This project illustrates the Central Limit Theorem using Monte Carlo simulation.

## Objective
Demonstrate that the distribution of sample means tends toward a normal distribution.

## Method
- Generate 10,000 samples of size 30 from an exponential distribution
- Compute the sample means
- Visualize the histogram of sample means

## Tools
Python, NumPy, Matplotlib

## Result
The histogram (clt_histogram.png) shows the convergence to a normal distribution.
